**The Tag Protocol** es una plataforma de vanguardia diseñada para la auditoría estética y financiera de la moda de alto nivel. En un mundo donde el estatus se comunica visualmente, nuestra misión es proporcionar una herramienta de valuación precisa y validación social en tiempo real.

---

## 💎 La Visión
No somos una red social más; somos un **sistema de arbitraje de estilo**. **The Tag** permite que cada prenda sea un activo digital cuantificable:
* **Transparencia de Estatus:** Valuación real de activos de lujo sobre imagen.
* **Flex Score:** Algoritmo dinámico que calcula el impacto financiero del outfit completo.
* **Social Audit:** Una democracia de estilo donde la comunidad decide qué es tendencia y qué es pretencioso.

## 🚀 Hoja de Ruta (Roadmap)
- [x] **Dynamic Tagging Engine:** Interfaz interactiva de alta precisión para etiquetado.
- [ ] **Camera Integration:** Captura directa desde dispositivos móviles para registro instantáneo.
- [ ] **Luxury Database:** Base de datos global de precios y marcas de alta gama.
- [ ] **Elite Leaderboard:** Ranking mundial de los outfits con mayor valor de mercado.

## 🛠️ Stack Tecnológico
* **Frontend:** HTML5 / CSS3 / Tailwind CSS (Estética Royal Navy & Gold).
* **Intelligence:** JavaScript Vanilla (Optimizado para baja latencia).
* **Infraestructura:** Despliegue global vía GitHub Pages.
